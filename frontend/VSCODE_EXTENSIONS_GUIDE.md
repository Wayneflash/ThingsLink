# VS Code Vue 扩展推荐（可视化编辑）

## 🎯 在搜索结果中，推荐这些扩展：

### 1. **Vue Language Features (Volar)** ⭐⭐⭐⭐⭐（必须安装）

**功能：**
- ✅ Vue 3 语言支持（你用的是 Vue 3）
- ✅ 智能提示、自动补全
- ✅ 语法高亮
- ✅ 组件跳转
- ✅ 错误检查

**说明：**
- 这个是 **Vue 3 的官方推荐扩展**
- 你的项目是 Vue 3，**必须安装这个**
- 不会可视化编辑，但是代码编辑的基础工具

**建议：** ✅ **安装这个**

---

### 2. **Preview.js** ⭐⭐⭐⭐⭐（可视化预览，推荐！）

**功能：**
- ✅ **可视化预览 Vue 组件**
- ✅ 直接在 VS Code 中预览组件
- ✅ 实时更新
- ✅ 支持 Vue 3

**说明：**
- 这个可以 **在 VS Code 中预览组件效果**
- 不是拖拽编辑，但是可以看到预览

**建议：** ✅ **安装这个（如果你想要预览功能）**

---

### 3. **TypeScript Vue Plugin (Volar)** ⭐⭐⭐⭐

**功能：**
- ✅ TypeScript 支持
- ✅ Vue + TypeScript 智能提示

**说明：**
- 如果你的项目是 JavaScript（不是 TypeScript），可以不用安装
- 从你的代码看是 `.js` 文件，**不需要这个**

**建议：** ⚠️ **不需要安装（除非你要用 TypeScript）**

---

### 4. **Vue 3 Snippets** ⭐⭐⭐⭐

**功能：**
- ✅ Vue 3 代码片段
- ✅ 快速生成代码模板

**说明：**
- 可以加快代码编写速度
- 不是可视化编辑工具

**建议：** ✅ **可选安装（提高效率）**

---

## ❌ 不推荐的扩展（对你没用）

- ❌ 这是 Vue 2 的扩展

### **Vue 2 Snippets** ⭐⭐
- ❌ 这是 Vue 2 的代码片段
- ❌ 你的项目是 Vue 3，**不需要这个**

- ❌ 这些不是可视化编辑工具
- ❌ 不符合你的需求

## 📋 安装步骤

### 在 VS Code 中：

1. **安装 Vue Language Features (Volar)**
   - 在扩展市场搜索：`Vue Language Features (Volar)`
   - 点击 "安装" 按钮
   - ⚠️ **如果已安装 Vetur，先卸载 Vetur**

2. **安装 Preview.js（可选，推荐）**
   - 搜索：`Preview.js`
   - 点击 "安装"

3. **安装 Vue 3 Snippets（可选）**
   - 搜索：`Vue 3 Snippets`
   - 点击 "安装"

---

## 💡 关于可视化编辑

**重要说明：**

这些 VS Code 扩展**不能像 Axure 那样拖拽编辑**，它们主要提供：
- ✅ 代码智能提示
- ✅ 语法高亮
- ✅ 组件预览（Preview.js）
- ❌ **不是拖拽式可视化编辑器**

**如果你想要真正的可视化编辑（拖拽式），需要：**
- 使用 **CodeSandbox** 在线编辑器
- 或者 **浏览器开发者工具**（F12）
- 或者我之前推荐的 **Page Edit 浏览器扩展**

---
