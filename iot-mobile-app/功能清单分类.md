# 物联网平台功能清单（Web端与APP端）

## Web端功能清单
### 1. 设备管理
- 设备注册与信息录入
- 设备状态监控（在线/离线）
- 设备分组与标签管理
- 设备详情查看（基础信息、数据记录）
- 设备批量导入/导出

### 2. 数据监控
- 实时数据仪表盘
- 历史数据查询与导出
- 数据趋势分析图表
- 自定义数据看板

### 3. 报警管理
- 报警规则配置
- 报警记录查询
- 报警通知设置（邮件/短信）
- 报警统计报表

### 4. 系统管理
- 用户角色管理
- 权限分配
- 操作日志审计
- 系统参数配置

## APP端功能清单
### 1. 核心功能
- 用户登录/登出
- 设备概览（总览统计）
- 设备列表（搜索/筛选）
- 设备详情（实时数据、控制）

### 2. 数据功能
- 实时数据卡片展示
- 历史数据图表
- 数据异常提醒

### 3. 报警功能
- 报警消息推送
- 报警列表查看
- 报警处理状态标记

### 4. 其他功能
- 个人中心
- 消息通知
- 多语言切换（可选）
