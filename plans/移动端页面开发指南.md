# 移动端页面开发指南

## 1. 设计原则
- **响应式布局**：使用 `flex` 或 `grid` 布局适配不同屏幕尺寸。
- **触摸友好**：按钮和交互元素大小不小于 48x48px。
- **性能优化**：避免过多 DOM 节点，减少重绘。

## 2. 适配 UniApp 的 Vue 页面
1. **页面结构**：
   - 使用 `uni-app` 提供的组件（如 `view`、`button`）替代 Web 专用组件。
   - 示例：
     ```vue
     <template>
       <view class="container">
         <text>移动端页面</text>
         <button @click="handleClick">按钮</button>
       </view>
     </template>
     ```
2. **样式适配**：
   - 使用 `rpx` 单位替代 `px`，自动适配屏幕密度。
   - 示例：
     ```css
     .container {
       padding: 20rpx;
     }
     ```

## 3. 6~7 个核心页面建议
1. **登录页**：简化表单，增加指纹/面部识别支持。
2. **设备列表页**：卡片式布局，支持下拉刷新。
3. **数据统计页**：适配 ECharts 移动端显示。
4. **设置页**：分组列表，简洁明了。

## 4. 测试与调试
- 使用 HBuilderX 的模拟器或真机调试。
- 检查不同分辨率的显示效果。
