# 核心页面原型说明

## 1. 登录页
- **功能**：用户认证（账号密码/指纹/面部识别）。
- **元素**：
  - Logo 和标题
  - 账号密码输入框
  - 登录按钮
  - 忘记密码/注册入口

## 2. 概览页
- **功能**：展示关键数据摘要。
- **元素**：
  - 设备总数/在线数卡片
  - 实时数据图表（ECharts 迷你图）
  - 报警摘要（最新 3 条）

## 3. 设备页
- **功能**：查看设备实时数据。
- **元素**：
  - 设备列表（卡片式，含状态指示灯）
  - 实时数据面板（数值/曲线图）
  - 控制按钮（如开关）

## 4. 历史数据页
- **功能**：查询历史数据。
- **元素**：
  - 时间选择器
  - 数据图表（ECharts 折线图）
  - 导出按钮

## 5. 报警页
- **功能**：管理报警信息。
- **元素**：
  - 报警列表（分页，含优先级标签）
  - 筛选条件（时间/级别）
  - 处理状态标记

## 6. "我的"页
- **功能**：用户设置和个人信息。
- **元素**：
  - 头像和用户名
  - 设置选项（通知/语言/主题）
  - 退出登录按钮

## 原型设计工具建议
- 使用 Figma 或 Adobe XD 快速绘制交互原型。
