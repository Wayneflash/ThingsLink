# 移动端核心页面原型设计

基于现有 Web 端功能，设计适配移动端的 6 个核心页面原型。

## 1. 登录页
- **设计要点**：
  - 简化 Web 版的动态背景，保留品牌 Logo 和标题。
  - 使用移动端友好的表单布局（垂直排列）。
  - 增加指纹/面部识别登录选项。
- **元素**：
  - Logo 和标题
  - 账号密码输入框
  - 登录按钮
  - 忘记密码/注册入口
  - 生物识别登录选项

## 2. 概览页
- **设计要点**：
  - 卡片式布局展示关键数据（设备总数、在线数等）。
  - 简化图表，使用迷你 ECharts 或进度条。
  - 底部导航栏快速切换页面。
- **元素**：
  - 数据摘要卡片（4个）
  - 报警摘要（最新3条）
  - 快捷操作入口

## 3. 设备列表页
- **设计要点**：
  - 列表项显示设备名称、状态和最后在线时间。
  - 支持下拉刷新和搜索。
  - 点击进入设备详情。
- **元素**：
  - 搜索框
  - 设备列表（卡片式）
  - 状态指示灯

## 4. 设备详情页
- **设计要点**：
  - Tab 切换实时数据/历史数据/控制命令。
  - 实时数据以简洁的数值卡片展示。
  - 历史数据支持图表和表格视图切换。
- **元素**：
  - 设备基本信息
  - 实时数据卡片
  - 历史数据图表
  - 控制命令按钮

## 5. 报警页
- **设计要点**：
  - 分级显示报警（严重/警告/提示）。
  - 支持按状态和级别筛选。
  - 点击查看报警详情和处理。
- **元素**：
  - 报警统计卡片
  - 筛选条件
  - 报警列表（带优先级标签）

## 6. "我的"页
- **设计要点**：
  - 显示用户头像和基本信息。
  - 提供设置选项和个人信息修改。
  - 退出登录按钮。
- **元素**：
  - 用户信息卡片
  - 设置选项列表
  - 退出按钮

## 交互说明
- 使用 `uni-app` 的导航组件实现页面切换。
- 表单控件适配移动端触摸操作。
- 图表使用 `echarts-for-uni-app` 组件库。
