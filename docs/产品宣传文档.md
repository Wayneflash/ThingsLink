## 平台简介

ThingsLink 是一款专为企业打造的物联网设备管理平台，已服务于制造、能源、农业、城市管理等多个行业的数百家企业客户。

与市面上的通用平台不同，ThingsLink 聚焦于工业级应用场景，提供从设备接入、数据采集、智能分析到远程控制的完整解决方案。平台支持百万级设备接入，每秒处理万级数据上报，核心业务接口响应时间小于200ms，满足企业级应用的高可靠性和高性能要求。

更重要的是，ThingsLink 支持私有化部署，所有数据存储在企业自有服务器，确保数据安全和业务自主可控。我们提供从部署实施、系统培训到运维支持的全生命周期服务，让企业真正拥有属于自己的物联网平台。

## 哪些人在使用 ThingsLink

### 工厂管理者：2分钟响应报警，电费降低18%

王总管理着一家拥有300台设备的工厂。以前设备出故障，往往是生产线停了才发现，每次损失数万元。

现在每天早上登录 ThingsLink 平台，数据概览大屏实时展示设备总数、在线状态、报警数量等关键指标。【截图1：数据概览大屏】24小时数据上报趋势图、设备分组分布、最新报警事件一目了然，全面掌握工厂运行态势。

上个月3号车间温度传感器触发报警，系统立即通过多渠道通知推送。王总在平台查看设备详情页，实时数据显示温度已达85℃，超出阈值15℃。【截图2：设备详情页实时数据】点击关联视频监控，实时画面确认现场情况，判断为设备异常升温。从报警触发到通知维修人员，整个响应时间不到2分钟，避免了生产中断。

更让王总满意的是能源管理功能。【截图3：能耗趋势分析页面】通过多维数据分析，发现2号车间夜班能耗较日班高出35%，深入排查发现是生产设备未按规范关闭。调整设备管理制度后，次月电费直接降低18%，年节省电费超60万元。

### 设备工程师：半天接入80台设备，办公室远程调试PLC

张工以前最头疼新设备接入，工厂里有PLC、机床、电表等不同设备，协议各不相同，每次都要写对接程序，少则一周多则一个月。更麻烦的是，不同厂商的数据格式不同，接入成本高。

现在通过 ThingsLink 自带的网关，网关支持广泛的工业协议，常见PLC、机床、电表、Modbus等协议都可以解析出来，最后转成标准的数据格式传输到云平台。【截图4：产品配置界面】在平台创建产品"智能温湿度传感器V2"，通过可视化界面定义产品属性（温度、湿度、电池电量）和控制命令（采集频率设置、休眠唤醒），配置完成后，80台设备通过批量导入功能半天全部接入平台。【截图5：设备列表页面】

更让张工意外的是，工厂里那批老设备也能接入。平台不仅支持 ThingsLink 自家的网关协议，也能兼容客户原有的老旧网关协议，可免费开发接入，老设备和新设备在同一平台统一管理，无需更换设备，保护了既有投资。

最让张工惊喜的是远程VPN通道功能。【截图6：远程VPN通道管理界面】以前给工厂PLC设备上传程序必须跑现场，现在在平台一键建立VPN安全隧道，直接在办公室就能通过远程连接上下载PLC程序、调试参数。上个月杭州工厂设备需要调试，张工在北京办公室远程搞定，节省了一次出差成本。

## 平台优势

### 1. 行业场景模板 + 数据深加工，不止于数据存储

ThingsLink 提供制造、能源、农业、城市管理等多个行业的场景模板，基于行业 Know-How 进行数据深加工。自动计算设备OEE、故障率、能耗趋势等关键指标，生成节能建议和故障预警，让数据真正产生价值，帮助企业优化运营决策。

### 2. 灵活多样的报警通知，确保问题不遗漏

支持站内信、邮件、短信、APP推送等多种通知方式，可根据报警级别灵活配置。严重报警可同时触发多种通知渠道，确保重要问题第一时间响应。

### 3. 全端适配，随时随地管理设备

提供 WEB 端和移动端双平台支持，数据和功能完全同步。WEB端适合数据分析和批量操作，移动端支持完整功能，报警实时推送，现场处理可拍照上传，实现真正的移动化管理。

### 4. 设备管理与视频监控一体化，数据与现场画面联动

将视频管理功能深度集成，设备报警时自动关联现场摄像头，一键查看实时画面。支持 GB28181 标准，兼容海康、大华等主流监控设备，支持历史录像回放，数据与画面结合，大幅提升问题判断和处理效率。

### 5. 支持私有化部署，数据安全自主可控

支持完全私有化部署在企业自有服务器，所有数据存储在企业内网，无数据外泄风险。企业完全掌控平台，不依赖外部服务，一次性投入，长期使用成本远低于云平台。提供标准化部署方案和完整的技术培训。

### 6. 支持丰富的OPENAPI，轻松集成到现有系统

提供完整的 RESTful API 接口，覆盖设备管理、产品管理、报警管理、数据查询、远程控制等核心功能。标准接口易于对接，支持 Token 认证，提供完整的 API 文档和示例代码，支持将设备数据无缝集成到ERP、BI、MES等现有业务系统中。

### 7. 网关协议兼容性强，保护客户既有投资

不仅支持 ThingsLink 自家的网关协议，也能兼容客户原有的老旧网关协议，可免费开发接入。企业无需更换现有网关设备，老设备和新设备可在同一平台统一管理，保护客户既有投资，降低系统迁移成本。这是市面上很多平台做不到的。
