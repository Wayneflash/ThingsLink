# 物联网平台原型设计

## 📁 文件清单

本目录包含物联网平台的完整 HTML 原型，采用苹果风格简洁设计。

### 🎯 核心页面

| 文件名 | 页面名称 | 功能说明 |
|--------|---------|---------|
| `index.html` | 导航页 | 原型总览，点击卡片进入各模块 |
| `login.html` | 登录页 | 用户登录（admin/admin） |
| `dashboard.html` | 数据监控 | 实时数据、统计卡片、趋势图表 |
| `product-manage.html` | 产品管理 | 产品列表、属性配置、命令管理 |
| `device-group.html` | 设备分组 | 树形分组结构、设备组织 |
| `device-list.html` | 设备管理 | 设备列表、在线状态、筛选 |
| `user-manage.html` | 用户管理 | 账户管理、角色分配 |
| `role-manage.html` | 角色管理 | 权限配置、菜单权限树 |
| `data-query.html` | 数据查询 | 历史数据查询、趋势图、导出 |

### 🎨 公共资源

| 文件名 | 说明 |
|--------|-----|
| `common-style.css` | 公共样式文件（苹果风格） |

## 📋 原型特点

### ✅ 设计原则
- **简洁美观**：参考苹果 UI 设计规范
- **详细备注**：每个页面都有原型说明和交互提示
- **可交互**：按钮、弹窗、Tab 切换等都可点击体验
- **统一风格**：所有页面使用公共样式，保持一致性

### 🎯 核心交互

#### 产品管理页面
- ✅ 创建产品弹窗
- ✅ 产品详情弹窗（Tab 切换：属性列表 / 命令列表）
- ✅ 属性和命令展示（支持任意自定义字段）

#### 设备分组页面
- ✅ 树形结构展示
- ✅ 点击分组查看设备列表
- ✅ 分组创建按钮

#### 设备管理页面
- ✅ 三级筛选（分组/产品/状态）
- ✅ 设备列表展示
- ✅ 在线/离线状态标识

#### 角色管理页面
- ✅ 左右布局（角色列表 + 权限配置）
- ✅ 菜单权限树（复选框）
- ✅ 权限保存按钮

#### 数据查询页面
- ✅ 多条件查询表单
- ✅ 趋势图占位
- ✅ 数据列表展示
- ✅ 导出功能按钮

## 💡 原型说明

### 关于数据
- 所有数据均为示例数据
- 交互通过 JavaScript 模拟
- 点击按钮会弹出提示说明后续操作

### 关于功能
- ✅ 页面跳转：完全可用
- ✅ 弹窗交互：完全可用
- ✅ Tab 切换：完全可用
- ⚠️ 表单提交：仅提示，不保存数据
- ⚠️ 图表渲染：占位区域，正式版用 ECharts

## 📝 后续开发

确认原型后，将按以下顺序开发：

1. **数据库设计**（已完成 init.sql）
2. **后端接口**（Spring Boot + MyBatis-Plus）
3. **前端页面**（Vue 3 + Element Plus）
4. **MQTT 集成**（EMQX + 消息处理）
5. **部署配置**（Docker Compose）

---

## 🔧 技术栈

**原型技术**：纯 HTML + CSS + JavaScript
**正式开发**：
- 前端：Vue 3 + Vite + Element Plus + ECharts
- 后端：Spring Boot 2.7 + MyBatis-Plus + Redis
- 数据库：MySQL 8.0
- 消息队列：EMQX (MQTT Broker)
- 部署：Docker + Docker Compose
